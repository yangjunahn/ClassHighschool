# 한성여고 고교학점제 수업 자료

이 저장소는 한성여자고등학교 고교학점제 수업에서 사용하는 데이터 및 인공지능 기초 실습 자료입니다.

- 강의 대상: 한성여자고등학교
- 강의 주제: 데이터 속성, 데이터 시각화, 기계학습 모델 기초
- 교재: 씨매스 「인공지능 기초」
- 실습 환경: Google Colab / Jupyter Notebook

## 포함된 파일

| 파일명 | 내용 |
| --- | --- |
| `3-1-1-DataIntro.ipynb` | 데이터 속성과 유형, 독립 변수와 종속 변수, K-pop 아이돌 데이터 속성 실습 |
| `3-1-2-Graph.ipynb` | 막대그래프, 선 그래프, 산점도, 히스토그램을 활용한 데이터 시각화 실습 |
| `3-2-1-KNNandLogistic.ipynb` | KNN과 로지스틱 회귀 분류 모델의 개념 및 비교 실습 |
| `3-2-2-TrainingData.ipynb` | 훈련 데이터와 테스트 데이터 분리, 모델 학습, 예측 결과 비교 실습 |
| `kpopidolsv3.csv` | 실습에 사용하는 K-pop 아이돌 데이터 |
| `.gitignore` | 가상환경 등 업로드하지 않을 파일 설정 |

## 수업 구성

### 3-1-1 데이터 속성과 유형

데이터 속성의 의미를 살펴보고, 기계학습에서 독립 변수와 종속 변수가 어떤 역할을 하는지 학습합니다. Kaggle K-pop 아이돌 데이터를 활용해 속성값, 수치형 데이터, 범주형 데이터, 결측값을 확인하고 데이터를 직접 추가해 봅니다.

### 3-1-2 데이터 시각화

막대그래프, 선 그래프, 산점도, 히스토그램 등 다양한 그래프를 활용해 데이터를 시각화합니다. 국가, 성별, 데뷔 연도, 키와 몸무게 관계 등을 분석하며 데이터 속성의 역할을 이해합니다.

### 3-2-1 KNN과 로지스틱 회귀

분류 모델의 개념을 학습하고, KNN과 로지스틱 회귀 알고리즘의 원리를 비교합니다. K-pop 아이돌 데이터의 키와 몸무게 속성을 활용해 산포도를 그리고, 새로운 데이터를 추가해 예측 결과를 확인합니다.

### 3-2-2 훈련 데이터와 테스트 데이터

기계학습 모델 구현 과정을 단계별로 학습합니다. 데이터를 훈련 데이터와 테스트 데이터로 나누고, KNN과 로지스틱 회귀 모델을 학습시켜 예측 결과와 성능을 비교합니다.

## 데이터 출처

실습 데이터는 Kaggle의 1700+ K-Pop Idols Dataset을 바탕으로 수업용으로 활용합니다.

- https://www.kaggle.com/datasets/nicolsalayoarias/all-kpop-idols

## 사용 방법

1. 이 저장소를 내려받거나 Google Colab에서 노트북 파일을 엽니다.
2. `kpopidolsv3.csv` 파일이 노트북과 같은 위치에 있는지 확인합니다.
3. 노트북의 셀을 순서대로 실행하며 실습을 진행합니다.
